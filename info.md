## Objeto Math

````
// arredondar numeros:
let num = 20.55245;
console.log(Math.floor(num)); // arrendoda sempre para baixo
console.log(Math.ceil(num)); // arrendoda sempre para cima
console.log(Math.round(num)); // arrendoda para o valor mais proxima acima ou abaixo de .5
````
